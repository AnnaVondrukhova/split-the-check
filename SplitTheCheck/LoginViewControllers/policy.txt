	Для получения информации о чеке приложение SplitTheCheck использует официальный сервис проверки кассовых чеков Федеральной Налоговой Службы России (далее - Сервис ФНС). Информация об использовании персональных данных содержится в Политике конфиденциальности Сервиса ФНС.

	Для использования приложения SplitTheCheck необходимо пройти регистрацию в Сервисе ФНС. Это можно сделать как в приложении SplitTheCheck, так и в официальном приложении ФНС России (далее - Приложение ФНС). Регистрация производится по номеру телефона (используется в качестве логина), на который высылается СМС с паролем. Пара «логин-пароль» позволяет проходить авторизацию в приложении SplitTheCheck, в Приложении ФНС, а также в любых других приложениях, использующих Сервис ФНС. 

	Если вы уже зарегистрированы в Сервисе ФНС, используйте имеющуюся пару «логин-пароль» для входа в приложение SplitTheCheck. 

	Если вы забыли пароль для авторизации в Сервисе ФНС, вы можете получить новый через приложение SplitTheCheck. На телефон, указанный в качестве логина, будет направлено СМС с новым паролем. Обращаем ваше внимание, что после получения СМС с новым паролем старый пароль не может быть использован для авторизации в Сервисе ФНС через приложение SplitTheCheck, Приложение ФНС и любое другое приложение, использующее Сервис ФНС. 

	Приложение SplitTheCheck использует предоставленный пользователем номер телефона только для авторизации в Сервисе ФНС. СМС направляются на указанный номер непосредственно Сервисом ФНС. Номер телефона хранится в приложении на устройстве пользователя и недоступен разработчикам приложения SplitTheCheck. 

	Приложение SplitTheCheck не передаёт данные пользователя третьим лицам и не использует облачные сервисы для хранения указанных данных и иной полученной информации.
